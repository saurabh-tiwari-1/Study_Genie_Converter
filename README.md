# Study_Genie_Converter [ HTML + CSS + JAVASCRIPT ]
"Study_Genie_Converter" is a free, all-in-one web tool for managing PDFs and images. It lets users merge, split, compress, convert, rotate, unlock, and watermark PDF files with one click. The platform also includes iGenerateIMG, an AI-powered image editor for enhancing and compressing photos. 

Key features:

Interactive PDF tools with file upload modals
User authentication (sign-up/login with demo storage)
Responsive design with modern gradients and animations
Contact section with email signup and legal links
Trust badges showing ratings and security features

Built with HTML, CSS, and JavaScript, it simulates a complete document productivity suite in a clean, user-friendly interface.
